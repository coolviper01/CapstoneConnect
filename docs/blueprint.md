# **App Name**: Capstone Connect

## Core Features:

- Consultation Scheduling: Advisers can schedule consultations, specifying semester, academic year, capstone title, block/group number, date, start/end time, and venue.
- Real-Time Discussion Editor: Advisers and students can collaboratively record notes during consultations.
- Attendance Tracking: Record attendance with digital signatures or typed names during consultations.
- PDF Report Generation: Automatically populate and download the 'Capstone Project Consultation Form' in PDF format with recorded details.
- Student Dashboard: Displays a list/calendar of scheduled consultations with capstone details and a 'Join Consultation' button.
- Teacher Monitoring Dashboard: Summary view of all consultations, ability to review/approve forms, and generate reports to analyze consultation insights. Also includes filters.
- Suggested talking points: Use the project details provided to suggest talking points that should be brought up during the consultation; present these suggestions to the advisor as a tool during their meetings

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire confidence and structure.
- Background color: Light gray (#ECEFF1) to ensure a professional, clean backdrop.
- Accent color: Teal (#009688) to highlight interactive elements and key actions.
- Body font: 'Inter', sans-serif, for clean and modern readability.
- Headline font: 'Space Grotesk', sans-serif, for clear section headings.
- Consistent use of minimalist icons for easy navigation and feature recognition.
- Responsive design for usability on desktops, tablets, and mobile devices.
- Subtle transitions and feedback animations to enhance user interaction and engagement.
